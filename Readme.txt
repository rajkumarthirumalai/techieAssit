TechieAssist - Technology Solutions Website

This project has been transformed from the Dewi Bootstrap template into a comprehensive technology solutions website called "TechieAssist".

Key Features:
1. About Us - Company information and mission
2. Services - 4 main service categories with detailed sub-services in tabs
3. Testimonials - Client feedback and reviews
4. Careers - Job opportunities and company culture
5. Contact Us - Multiple contact methods including live chat
6. Chat Functionality - Interactive chat widget for customer support

Services Offered:
- Web Development (Frontend, Backend, E-commerce, PWA)
- Mobile Apps (iOS, Android, Cross-platform)
- Cloud Services (AWS, Azure, DevOps, Infrastructure)
- AI & ML (Predictive Analytics, NLP, Computer Vision, Chatbots)

Technical Implementation:
- Responsive design using Bootstrap 5.3.3
- Interactive service tabs with detailed information
- Live chat widget with simulated bot responses
- Smooth animations using AOS library
- Custom CSS styling for enhanced user experience

Navigation:
- Home
- About Us
- Services
- Testimonials
- Careers
- Contact Us

The website is fully responsive and optimized for all devices.
